## Objects

- Unordered, key value pairs 

**Big O**.

* Insertion - O(1)
* Removal - O(1)
* Searching - O(N)
* Access - O(1)

**Objects Methods**.

* Object.keys - O(N)
* Object.values - O(N)
* Object.entries - O(N): ```[[key],[value]]```
* hasOwnProperty - O(1): return > ```true || false```
